# Game Unit Economics & Profitability Analysis

## Project Overview
This project evaluates the profitability and sustainability of different games by analyzing revenue, user acquisition costs, and overall unit economics.

---

## Analysis Focus

### 1. Revenue vs Cost
- Total revenue per game  
- Ad spend per game  
- Profit estimation (Revenue - Ad Spend)  

### 2. Unit Economics
- Revenue per user (ARPU)  
- Cost per user (CPU)  
- Revenue vs cost comparison  

### 3. Profitability Insights
- Which games are profitable  
- Which games are overspending  
- Opportunities to improve ROI
